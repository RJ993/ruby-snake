April 3rd 2025 --> Demo version. Part of what took me so long is the getch, timeout, terminal, and ANSI escape sequences rabbit holes. At first, I was skeptical of the getch method simply because there is not much of an explanation of how it works. However, I figured out that it essentially puts the terminal into raw mode and then ensures that it is put back into cooked mode after getting a character (I learned that the terminal has modes. Did not know that before embarking on this journey). Another rabbit hole was the two timeouts that Ruby has. One being the, very well documented, module Timeout. I think that it creates another thread and then uses that thread to keep track of the elapsed time and if that time is elapsed, a Timeout::Error occurs. Then there is the IO Timeout, the less documented one (the one I used). How it works is that there is a timeout variable baked into the IO class. You can set any IO streams (i.e. $stdin, sockets) timeout variable to equal a certain numbers of seconds. If that amount of seconds pass, a IO::TimeoutError occurs. So yeah, it was a long journey.
